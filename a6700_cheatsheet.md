# Sony a6700 — Field Settings Cheat Sheet

## Quick-Lock Patterns (Commit These First)

| Mnemonic | Rule |
|----------|------|
| **Sunny 16 (digital)** | ISO 100 · f/8 · 1/250s → tweak ±1 stop |
| **Golden ≠ Blue** | Golden = warm WB (6500K+). Blue = cool WB (4800K−) |
| **56mm = Portrait king** | Always shoot wide-open (f/1.4–f/2.8) unless landscape |
| **18mm = Landscape anchor** | f/8–f/11, focus ⅓ into frame |
| **Video SS rule** | Shutter = 2× frame rate (24fps→1/50, 60fps→1/125) |
| **ISO ceiling** | Photo: 6400. Video: 3200 (S-Log3 base 800/2500) |

## Lazy-Load Memory System (How to Memorise This)

### Layer 1 — Lens Identity (Non-Negotiable)
| Lens | Aperture Range | Best For | Weakness |
|------|---------------|----------|----------|
| **18–55mm f/3.5–5.6** | f/3.5–f/22 | Landscapes, walk-around video | Low light (use tripod at night) |
| **56mm f/1.4** | f/1.4–f/16 | Portraits, subject isolation, night | No zoom, tight framing |

### Layer 2 — Aperture by Scenario (The f-stop cheat)
| Scenario | 18–55mm Aperture | 56mm Aperture |
|----------|-----------------|----------------|
| 📷 Landscape | f/8–f/11 (sharpest) | f/4–f/8 |
| 📷 Portrait | f/3.5–f/5.6 (max available) | f/1.4–f/2.8 |
| 🎥 Cinematic | f/8–f/11 (deep DOF) | f/2.0–f/4 |
| 🎥 Portrait Vid | f/3.5–f/5.6 (max available) | f/1.4 (dreamy bokeh) |

### Layer 3 — ISO by Light (Think in Stops)
| Light Level | ISO Range | When |
|-------------|-----------|------|
| ☀️ Full Sun | 100 | Midday, Golden Hour |
| ☁️ Overcast | 200–800 | Cloudy daytime |
| 🌆 Dusk/Dawn | 400–1600 | Blue hour, twilight |
| 🌙 Night (56mm) | 800–3200 | f/1.4 saves you |
| 🌙 Night (18–55mm) | 1600–6400 | Tripod essential below 1/30s |

### Layer 4 — WB by Temperature (K = Kelvin)
| Time | Sun / Cloud / Rain | K Value |
|------|-------------------|---------|
| 🌅 Golden Hour | Any weather | 5500–6500K |
| ☀️ Midday | Sunny → Cloudy → Rain | 5200K → 5600K → 5800K |
| 🌆 Blue Hour | Any weather | 4200–5000K |
| 🌙 Night | Any weather | 3400–4000K |

### Layer 5 — Mode Decision Tree
```
Am I shooting Photo?
 ├─ Moving subject?   → Portrait → A-Priority (control DOF)
 └─ Still scene?      → Landscape → A-Priority (control aperture)

Am I shooting Video?
 ├─ Story/film look?  → Cinematic → Manual (full control)
 └─ Person talking?   → Portrait Vid → Manual (fixed shutter at 1/50s)
```

### Layer 6 — Focus Rules
| Subject | Photo | Video |
|---------|-------|-------|
| Person (portrait) | Eye-AF | Eye-AF |
| Landscape / city | MF at ∞ or hyperfocal | MF (no hunting) |
| General static | AF-S | MF |
